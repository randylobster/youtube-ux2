## 💡 About

Mass YouTube Video Downloader is designed for speed, reliability, and quality.  
It supports **playlists, channels, Shorts, and private videos**, while offering **lossless audio extraction** and **smart queue management**.  
All wrapped in a lightweight desktop UI built with **Python** and **PySide6**.

---

## 🌟 Key Features

### 🛠️ Core Features
| Feature | Description |
|----------|-------------|
| 🎥 **Universal YouTube Support** | Download from any YouTube link — videos, playlists, channels, or Shorts |
| 💾 **Multiple Formats** | MP4, MP3, M4A, WAV, FLAC, AAC, OPUS, and VORBIS |
| 🧠 **Smart Playlist Organizer** | Auto-sorts videos into playlist-named folders |
| 🕒 **Scheduler** | Automatically start downloads at a specific time |
| 🔊 **Audio Quality Control** | Choose bitrates (128–320k) or use “Best” mode for lossless quality |
| 📜 **Download History** | View and manage all past downloads |
| 🖱️ **Drag & Drop** | Add URLs directly from your browser |
| 🪶 **Lightweight** | Built-in FFmpeg and minimal dependencies |

---

### ⚙️ Advanced Features
- **Batch Processing** – Queue and process multiple YouTube links at once  
- **Zero-Loss Extraction** – Smart copy mode for original AAC/OPUS/M4A audio  
- **Profile Management** – Save user settings and profiles  
- **Import / Export Profiles** – Migrate between devices easily  
- **Proxy & Bandwidth Control** – Manage speed or network preferences  
- **Auto-Updater** – Installs the newest version automatically  
- **System Tray Integration** – Run silently in the background  

---

### 🎨 User Experience
- Dark & Light themes  
- Download notifications (complete, failed, canceled)  
- Modern animations and smooth UI transitions  
- Detailed error logs  
- Searchable history and filtering

### 🔧 Technical Features
- Built with **Python 3.13+**, **PySide6**, and **FFmpeg**  
- Modular architecture: `core/`, `ui/`, `tests/`  
- FFmpeg auto-detection & fallback setup  
- Fully compatible with **yt_dlp** for metadata and format detection

2. Run the installer or extract the `.zip`  
3. Launch **YouTube Downloader.exe** — all dependencies (including FFmpeg) are included  

## 🔧 Usage

### Basic Usage
```bash
# Launch the app
python main.py
```

### Key Features Usage
- Configure your profile in the **Settings**  
- Use the **MP4** or **MP3** tabs to download YouTube videos or extract audio  
- Add multiple downloads to the queue  
- Schedule downloads in advance  

### Tips & Tricks
- Drag & drop for quick URL addition  
- Enable system tray for background operation  
- Use scheduler for off-peak downloads  
- **Best Audio Quality**: Preserve Original + 320k bitrate  
- **Lossless Audio**: Use M4A/FLAC with copy mode  
- Export your profile for migration
